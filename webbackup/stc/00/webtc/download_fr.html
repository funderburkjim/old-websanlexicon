Dictionnaire sanscrit-français Stchoupak : Téléchargements

Pour d'autres dictionnaires sanskrits, s'il vous plaît visitez la page d'accueil de COLOGNE Lexiques sanskrits.

 

  1. stctxt.zip
    • stc_orig.txt La numérisation d'origine.
    • stc_orig_utf8.txt La numérisation d'origine en encodage UTF-8.
    • stc.txt La numérisation en fichier texte.
    • stc-meta.txt Documentation des conventions de codage de stc.txt
    • stcheader.xml Détaille les autorisations d'utilisation et d'autres renseignements concernant cette édition.
    • CDSL.pdf Description du projet sanskrit-Lexique/
  2. stc_bookmark.pdf (faites un clic droit "Enregistrer sous" pour télécharger) Fichier pdf des images numérisées des pages du dictionnaire. Des signets (notant le premier mot de chaque page) permettent d'accéder aux pages.
  3. stcxml.zip
    • stc.xml Une version XML de stc.txt, par mots-clés
    • stc.dtd Définition du type de document pour stc.xml
    • programs Programmes logiciel qui crée stc.xml à partir de stc.txt
  4. stcweb.zip Comment construire une version hors ligne de l'affichage Web du dictionnaire. Comprend des images numérisées.
  5. stcweb1.zip Comment construire une version hors ligne de l'affichage Web du dictionnaire. Aucune image numérisée.
