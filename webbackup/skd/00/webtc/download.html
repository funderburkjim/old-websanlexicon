Sabda-kalpadruma

Downloads

  1. skdtxt.zip
    • skd_orig.txt The original digitization, in cp1252 encoding.
    • skd_orig_utf8.txt The original digitization, in utf-8 encoding.
    • skd_v0.txt Body of dictionary, in utf-8 with Devanagari in SLP1, and nuktas removed.
    • skd.txt The body of the dictionary, with some corrections.
    • skd-preface.txt The dictionary preface.
    • skd-meta.txt Documentation of the coding conventions of skd.txt
    • skdheader.xml Description of licensing and other details of this edition
  2. skd1_bookmark.pdf (right-click 'save as' to download) Volume 1 of the scanned images to which the digitization corresponds. Bookmarks allow access of pages via the first word on each page.
  3. skd2_bookmark.pdf (right-click 'save as' to download) Volume 2 of the scanned images.
  4. skd3_bookmark.pdf (right-click 'save as' to download) Volume 3 of the scanned images.
  5. skd4_bookmark.pdf (right-click 'save as' to download) Volume 4 of the scanned images.
  6. skd5_bookmark.pdf (right-click 'save as' to download) Volume 5 of the scanned images.
  7. skd_title.pdf (right-click 'save as' to download) Title/preface pages from vol 1, scanned images.
  8. skdxml.zip
    • skd.xml An xml version of skd.txt, by headword
    • skd.dtd Document type definition for skd.xml
    • programs Software by which skd.xml is created from skd.txt
  9. skdweb.zip How to construct an offline version of the web displays of the dictionary. This download contains scanned images (500+MB)
  10. skdweb1.zip How to construct an offline version of the web displays of the dictionary. This download contains no scanned images.
