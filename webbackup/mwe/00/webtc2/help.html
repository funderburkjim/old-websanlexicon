Help for Monier-Williams English-Sanskrit Advanced Search

This program does partial word searches for English head words or other text words and for Sanskrit text words in the dictionary. For the basic usage, type a word in the text input box next to 'Sanskrit text word:' or 'English text word/Headword:', set the search paramenters, and the maximum and output parameters, and click the button. A message in the left display pane indicates the program is working. When the request is finished, output appears in the left display pane.

Instead of clicking the button, one can press the key while the cursor is in the text input box. (NOTE: Currently, this is known to work properly only when using the Firefox or IE7 browsers with the Windows XP operating system. It is known to work improperly for the Safari browser on the Macintosh; however, the 'Search' button seems to work ok in this case.)

The data in the left pane presents the underlying headwords where matches occur; if you are searching for a Sanskrit text word, the matching word appears in parentheses next to the headword. The head word is clickable. If clicked, the full record for the word appears in the right-hand pane. All Sanskrit words appear according to the setting of the 'output' parameter.

Various controls allow some degree of tuning of the search process and of the output format.

  • exact/prefix/suffix/substring (Sanskrit or English word). This determines where within a possibly longer word the match occurs.
  • HK/SLP1/ITRANS (Sanskrit word): the transliteration scheme in which you input the Sanskrit word.
  • Maximum: this number determines the maximum number of matches per search. Bigger numbers will take longer, of course.
  • output: This allows a choice of how to represent the Sanskrit words in the output, in each pane.
  • button: This button is available after a search in which the maximum number of matches did not exhaust the dictionary. In other words, there may be more matches. Clicking the button continues the search, from the point at which the prior search ended.

  • Press again
  • OR, Re-enter the search parameters, then press again.
  • OR, Press the 'reload' button for your web browser, then re-enter the search parameters, then press again.
