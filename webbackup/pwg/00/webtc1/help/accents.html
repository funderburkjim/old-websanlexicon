Accents

This document aims to describe Sanskrit accents appearing in the Grosses Petersburger Wörterbuch of Böhtlingk and Roth (PWG).

Sanskrit accents occur throughout the text, and are coded in the digitization. The display of accents takes the form of various decorations of the text. The displays where accents are available have an option to 'Ignore Accents' (the default) or 'Show Accents'. When the 'Ignore Accents' option is in force, the displays do not show the accents.
When a display is configured by the user to show accents, then the visual form of the accent decorations in the displays further depends on the user's choice for displaying textual Devanagari.
Finally, proper display of the accents with Devanagari depends on the use of web fonts (discussed below).

The interested reader may find extensive discussion of various representations of Sanskrit accents in the book Linguistic Issues in Encoding Sanskrit by Peter M. Scharf and Malcolm D. Hyman, published in 2011. A pdf version of the book may be found at sanskritlibrary.org, in the Publications section. Our display of accents aims to be consistent with those shown on page 162ff.

There are three types of accents that occur in PWG:

  • udātta, the acute accent , a high or sharp tone
  • anudātta, the grave accent, not elevated or raised
  • svarita, a kind of mixed tone , produced by a combination of high and low tone

Bohtlingk and Roth use the common system of marking accents with the vertical bar above a svarita vowel and a horizontal bar beneath an anudAtta vowel when they display continuous texts. On individual words such as headwords, however, they place a Devanagari 'u' above the udAtta vowel and vertical bar above an independent svarita vowel. The digital encoding of Bohtlingk and Roth's work marked udAttas, anudAttas, and svaritas, but did not distinguish independent from dependent svaritas. The current display, in both Devanagari and Roman, simply translates the encoding marks into available characters; it does no accent sandhi or interpretation. The characters selected are appropriate for isolated words, i.e. headwords in particular but use the uncommon Kashmiri marks: vertical bar above an udAtta vowel, hook above an independent svarita vowel. However, the same marks have been used admittedly inappropriately for the continuous Vedic texts. Please bear with us while we improve the display of the continuous Vedic passages.

With Devanagari display of Sanskrit, the accents are represented by the following unicode points:

  • The udātta accent uses unicode code point \u0951, DEVANAGARI STRESS SIGN UDATTA.
  • The anudātta accent uses unicode code point \u0952, DEVANAGARI STRESS SIGN ANUDATTA.
  • The svarita accent uses unicode code point \u1ce0, VEDIC TONE RIGVEDIC KASHMIRI INDEPENDENT SVARITA.

Web Fonts

As of now (2014), default Unicode fonts available on personal computers do not include representations for the Unicode code points used for all three accents in Devanagari text. Fortunately, there do exist fonts designed for the display of Devanagari text that do represent these accents. Two fonts that we are aware of are:

  • praja_Vedic font designed by Peter Freund; this is a non-free font which may be purchased at the Praja font order page
  • Siddhanta, a free font designed by Mihail Bayaryn.
    See also.

Recently, Marcis Gasuns brought to our attention the web technology known as Web Fonts. For an appropriately coded web page, a web browser may be instructed to download a font to be used within the web page. Most modern web browsers support web fonts.
We have chosen to use the Siddhanta font as a web font for display of Devanagari with accents.

This usage requires no font installation by the user; it just works. The first time that a web page uses a web font, the browser must download the font from our server; this typically results in a short delay for rendering all or part of the page. However, the browser typically caches a downloaded font, so that subsequent uses of the web page (such as to look up several more words) result in no delay.

For PWG, web fonts are also used in the display of other text.

  • The Old Standard font designed by Alexey Kryukov is used for the display of most text.
  • The Charter Indologique Capital font is used for the display of the larger and smaller capital letters in the literary source references.
